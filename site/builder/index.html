Explanation

Firmware 21.0.0+ introduced a 10MB deficit to the Applet Pool. To compensate, the core interception layer relies on the memlet helper module to dynamically steal an 8192 KB reserve from the System Pool to compile LayeredFS game modifications. If background daemons consume this space, the system will suffer a Data abort (0x101) upon game launch.
